HTML

  • HTML gives structure to a web page.
  • Use the "!" to create an automatic structure.
  • Parts of the HTML are called elements.

CSS

  • CSS adds style to a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Step 1: git checkout -b When you want to create a new branch. This only makes the branch on your computer. Step 2: git add -A Any changes that you have made and have saved to your computer. If you run this then you are saying you are ready to put them in a place just before turning and pushing in all your changes. Step 3: git commit -m “Write a description of what you are committing” Running this will like submitting the papers to the teacher. Step 4.1: git push --set-upstream origin When its the first time that you are trying to push changes to GitHub. Doing this means that you are creating the branch online on GitHub, and you are pushing your changes to it. Step 4.2: git push If you have already ran step 4.1 once then you can now run this one. Any time after your second save.
